#  CloudShip – End-to-End DevOps CI/CD Pipeline

##  Project Overview
CloudShip is a production-style DevOps project demonstrating how modern companies build, test, containerize, deploy, and monitor applications using cloud-native tools.

## Tech Stack
- AWS (EC2, VPC, IAM, EKS)
- Docker
- Kubernetes
- Terraform
- GitHub Actions (CI/CD)
- SonarQube
- Prometheus & Grafana

## Project Goals
- Build real CI/CD pipeline
- Automate infrastructure using Terraform
- Deploy containerized apps on Kubernetes
- Implement monitoring and observability
- Follow production-grade DevOps practices
